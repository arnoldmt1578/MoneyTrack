# MoneyTrack

## Descripción
MoneyTrack es una aplicación móvil para Android que permite registrar y controlar ingresos y gastos personales, ayudando a los usuarios a administrar mejor sus finanzas.

## Tecnologías utilizadas
- Android Studio
- Java
- SQLite

## Arquitectura
MVC (Modelo - Vista - Controlador)

## Integrante
- Arnold Mejía Tineo

## Estructura del proyecto

MoneyTrack
│
├── model
├── view
├── controller
├── database
└── utils
